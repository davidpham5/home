About Me

Hi. I'm David Pham. I'm a frontend designer, some times public speaker, and areo-pressed coffee enthusiast currently residing in Washigton DC. I help people create exceptional interfaces through human centered design.

I have a preference for Anonymous Pro as my fixed width type for my code editor, Sublime Text 3 (former Coda 2 user). I taught myself how to code and program. Ever since I graduated school, I'm drawn to people who built things. It was not until a few years I found my path.

I've kept a small portfolio to show off some things I did in the past. I usually cringed at how inexperience I was. I've learned to embrace that cringe feeling now. Without hard work, failures, and long sighs, I wouldn't have become a designer. When you're done looking around, lets get some coffee and talk shop.
